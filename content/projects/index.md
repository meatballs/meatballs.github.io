# Projects

Here are some of the open source projects I've worked on:

## Anvil Extras

A library of extra components and modules for use in creating an Anvil app.

- **Demo**: [anvil-extras-demo.anvil.app](https://anvil-extras-demo.anvil.app/)
- **GitHub**: [github.com/anvilistas/anvil_extras](https://github.com/anvilistas/anvil_extras)
- **Technologies**: Anvil, Python

## Amoni

A command line utility for local development of Anvil apps.

- **Documentation**: [amoni.readthedocs.io](https://amoni.readthedocs.io/en/latest/)
- **GitHub**: [github.com/anvilistas/amoni](https://github.com/anvilistas/amoni)
- **Technologies**: Python, CLI

## Axelrod Python

A Python game theory library for the study of the Iterated Prisoners Dilemma.

- **Documentation**: [axelrod.readthedocs.io](https://axelrod.readthedocs.io/en/latest/)
- **GitHub**: [github.com/axelrod-python/axelrod](https://github.com/axelrod-python/axelrod)
- **Technologies**: Python, Game Theory

## SQL for Python Programmers

A tutorial covering basic SQL for those with a background in Python.

- **Start Tutorial**: [owencampbell.me.uk/sql_python_tutorial](https://owencampbell.me.uk/sql_python_tutorial/)
- **GitHub**: [github.com/meatballs/sql_python_tutorial](https://github.com/meatballs/sql_python_tutorial)
- **Technologies**: Python, SQL, Tutorial
